Click the button below to login using OAuth2 Authorization Server

Login with My OAuth2 Server

How it works:

  1. Click the login button above
  2. You'll be redirected to the Authorization Server (port 9000)
  3. Login with: staff/staff or leader/leader
  4. Grant the requested permissions
  5. You'll be redirected back to this application with an authorization code
  6. The application will exchange the code for an access token
